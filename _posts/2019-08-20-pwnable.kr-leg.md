# Toddler's Bottle 
## 7. leg

leg.c  
```
#include <stdio.h>
#include <fcntl.h>
int key1(){
	asm("mov r3, pc\n");
}
int key2(){
	asm(
	"push	{r6}\n"
	"add	r6, pc, $1\n"
	"bx	r6\n"
	".code   16\n"
	"mov	r3, pc\n"
	"add	r3, $0x4\n"
	"push	{r3}\n"
	"pop	{pc}\n"
	".code	32\n"
	"pop	{r6}\n"
	);
}
int key3(){
	asm("mov r3, lr\n");
}
int main(){
	int key=0;
	printf("Daddy has very strong arm! : ");
	scanf("%d", &key);
	if( (key1()+key2()+key3()) == key ){
		printf("Congratz!\n");
		int fd = open("flag", O_RDONLY);
		char buf[100];
		int r = read(fd, buf, 100);
		write(0, buf, r);
	}
	else{
		printf("I have strong leg :P\n");
	}
	return 0;
}
```
main 함수를 보면 key1()+key2()+key3() 값을 입력해야한다.

컴퓨터의 명령어 fetch 사이클은 동시에 이뤄진다.  

| T1    | T2     | T3      | T4      | T5      |  
|-------|--------|---------|---------|---------|  
| FETCH | DECODE | **EXECUTE** |         |         |  
|       | FETCH  | **DECODE**  | EXECUTE |         |  
|       |        | **FETCH**   | DECODE  | EXECUTE |  
0x00008cdc를 수행함과 동시에 0x00008ce0를 디코드하고  0x00008ce4를 다음 명령어로 가리킨다.  
때문에 pc = 0x00008ce4
